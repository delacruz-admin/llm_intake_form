# Product Overview

This is the ARB (Architecture Review Board) Intake System for Cooley LLP's Technology Infrastructure team. It manages the lifecycle of technology infrastructure requests — from submission through triage, discovery, and completion.

The system includes:
- An AI-assisted chat intake flow that guides requestors through submitting infrastructure requests (using Anthropic Claude via browser-side API calls)
- A dashboard for tracking and managing all ARB requests with filtering, sorting, and detail views
- An AWS environment provisioning request form (multi-step wizard)
- A UI style guide documenting the design system

The domain is enterprise IT governance at a law firm. Requests cover things like new services, platform migrations, compliance initiatives, AI/ML pipelines, and vendor integrations.
